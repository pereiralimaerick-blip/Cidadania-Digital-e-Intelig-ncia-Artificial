# Cidadania-Digital-e-Intelig-ncia-Artificial# Portal Cidadania Digital e Inteligência Artificial 🌐🤖

Este site é um portal interativo de conscientização comunitária desenvolvido para ensinar os cidadãos a identificarem mídias manipuladas por IA (*deepfakes*) e alertar sobre os riscos da desinformação automatizada na sociedade moderna.

## 🛠️ Tecnologias Utilizadas
* **HTML5 Semântico:** Estruturação organizada e acessível.
* **CSS3 Customizado:** Design responsivo com Flexbox, efeitos interativos e variáveis de ambiente.
* **JavaScript Puro (Vanilla JS):** Manipulação dinâmica do DOM para validação de formulários e acessibilidade.

## ⚙️ Funcionalidades Principais
1. **Modo Escuro Interativo:** Botão de acessibilidade que alterna instantaneamente a folha de estilos global do site para conforto visual.
2. **Quiz Anti-Desinformação:** Sistema de validação lógica dinâmico que lê as respostas do usuário, calcula a performance e emite um veredito personalizado na tela sem atualizar a página.
3. **Layout Fluido:** Interface 100% responsiva adaptável para smartphones, tablets e desktops utilizando CSS Media Queries.
